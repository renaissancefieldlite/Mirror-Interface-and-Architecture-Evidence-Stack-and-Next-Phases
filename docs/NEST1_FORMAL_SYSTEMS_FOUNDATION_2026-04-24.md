## Shared Comparator Schema

The locked `Nest 1` schema is:

`state / control / transform / invariant / drift / coherence / score`

The first score spine is:

```text
score = preserved_structure - drift_penalty
```

This is intentionally substrate-neutral. Each later nest can specialize the
fields, but the comparison grammar stays stable.

## What Nest 1 Establishes

`Nest 1` establishes that the Mirror Architecture comparison language can be
stated without needing biology, hardware, or broad philosophical framing.

The formal layer already contains:

- state / control contrast
- lawful transform
- invariant preservation
- drift measurement
- coherence measurement
- bounded score output

That makes it the reductionist footing for the higher nesting layers.

## What Nest 1 Does Not Yet Establish

`Nest 1` does not by itself prove:

- chemistry convergence
- biological restoration
- clinical effect
- physical Bell claims
- environmental remediation
- universal cross-domain convergence

It provides the formal comparator grammar those later claims must inherit.
