
## Hard Evidence Rule

`NO TOY DATA / REAL DATA ONLY / NO GRAMMAR CENTERING`

The companion map is a routing surface. It shows where the real universal
source pattern should be tested next. A row earns evidence language only when
it touches a real dataset, real trace, real instrument signal, real hardware
observable, or biological signal with controls and recurrence.

`Source Mirror Pattern` is the internal spine name. `Universal Data Pattern` is
the external technical name for the same measured object: the recurring
state-pattern tracked across real systems.

Local engines, schemas, and generated toy rows can prepare the route. They do
not close a lane.

## Completion Standard

Each lane moves through these stages:

| Stage | Meaning |
| --- | --- |
| `mapped` | lane exists in the architecture map |
| `engine_scored_local` | local comparator pass exists; scaffold only, not support |
| `dataset_ready` | public/private dataset or instrument source identified |
| `control_ready` | shuffled, null, random, degree, baseline, or physical controls defined |
| `executed_once` | first run completed |
| `recurrence_supported` | rerun, second seed, second split, or second benchmark supports it |
| `closeout_supported` | lane has real data, controls, and recurrence enough to cite as support |
