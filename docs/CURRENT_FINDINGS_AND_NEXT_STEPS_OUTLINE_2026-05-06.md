Oxygen / Redox, Minerals, Electrochemistry, Materials / Semiconductors, and

finding is endpoint discipline: parent disappearance alone earns zero. Product
distribution, retained burden, bad descendants, defluorination /
mineralization quality, toxicity proxy, and environmental fate carry the

support surface. A `100,000`-row Matbench / Materials Project-derived run
scored Pearson `0.858577`, R2 `0.737141`, RMSE improvement `0.487314`, and
cleared shuffled-target controls with p `0.001996`. Composition /

The next materials subgate moves to targets that expose geometry and
nanoscale response directly: cell-geometry, crystal stability, defect, band,
phonon, dielectric / optical, 2D material, and nanotech confinement rows.

`Nest 2D Allostery` has real benchmark footing. AlloBench / PDB / P2Rank runs
show that pocket and path structure can carry biological signal, but the
