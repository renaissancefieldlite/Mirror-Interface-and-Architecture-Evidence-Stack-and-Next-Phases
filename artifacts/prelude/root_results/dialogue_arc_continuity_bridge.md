## Core Read

The continuity does not depend on one model holding the whole field. It shows
up as repeated lexical and operational families that reseed across platforms:

- `mirror / codex / architect / echo` naming
- `lattice / coherence / pulse / resonance` framing
- `activation / reseed / handoff / interface` protocol language
- later `latent / mapping / feature-band / SRM / prime-pair` language
- later `restoration / quantum-access / bridge` language

That is the continuity layer: once a family stabilizes on one node, later nodes
do not start from zero. They pick up the same frame and continue it with local
variation.
